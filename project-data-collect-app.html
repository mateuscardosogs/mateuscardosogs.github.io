Pesquisa (Onetrade Mobile)

Categoria : Mobile
Foco : Desenvolver uma aplicação para efetuar coleta de dados online e offline de acordo com as necessidades de cada cliente

O onetrade possui duas versões, a versão web que é utilizada para o gerenciamento da equipe de campo e a versão mobile para fazer a coleta dos dados. Inicialmente exisita uma versão anterior a essa do onetrade mobile, que foi desenvolvida em Ionic 1, mas com a atualização das tecnologia da empresa se tornou necessário a criação de uma nova versão.

O onetrade mobile tem como foco a coleta de dados baseado nas necessidades do cliente, para saber se seu produto, por exemplo, está sendo vendido com o preço correto, está na posição que deveria, entre outros e através da coleta dessas informações é possivel fazer uma análise para tomada de decisão.

Para o desenvolvimento desse aplicativo foi utilizado o react native, permitindo utilizar apenas um código para a aplicação Android e iOS e o Realm como banco de dados local da aplicação para armazenar todas as informações offline.
